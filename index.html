# Lost Cars Listing Website A mobile-first web platform to help list and search for lost cars using license plate or chassis numbers. **Admin-controlled:** Only you (the admin) can upload, edit, or remove listings, upload car images, and manage contact information. **Privacy-first:** Sensitive details and images are kept private, and your contact number is only revealed if a car is found. ## Features - **Add Lost Cars** (Admin-only): Upload car details and images. - **Public Search:** Anyone can search for a car by license plate or chassis number. - **Private Images/Contacts:** Only admin can view/edit images and sensitive info. - **Contact Workflow:** If a car is found, user submits a form, admin verifies, then shares contact number. - **Mobile-Friendly:** Optimized UI for use on any phone. ## Tech Stack - **Frontend:** React + Tailwind CSS (for fast, responsive mobile UI) - **Backend (suggested):** Firebase, Supabase, or similar (not included in this starter) - **Hosting:** Vercel, Netlify, or GitHub Pages - **Version Control:** GitHub ## Getting Started 1. **Clone this repo** 2. `npm install` 3. `npm start` (or use your favorite mobile-friendly IDE or deploy directly) 4. Begin customizing! ## Folder Structure ``` src/ components/ AdminPanel.js CarForm.js CarList.js SearchBar.js CarDetails.js pages/ Home.js Admin.js App.js index.js public/ index.html README.md ``` ## Admin Workflow 1. Log in as admin. 2. Add new lost car details, upload images (private). 3. Listings appear in search (public does NOT see images or contact). 4. When a user claims to have found a car, they send a form → admin verifies → shares contact if confirmed. ## License MIT ---
